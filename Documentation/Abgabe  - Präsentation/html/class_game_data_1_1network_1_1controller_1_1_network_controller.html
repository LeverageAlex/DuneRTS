Detailed Description

Base class for the network controller (in the clients and server). The network controller manage all receiving and sending of messages to the recipient

The network controller is used for handling the received and messages and the ones, which need to be send. Therefore is has two methods and handling the messages and hold the references to the connection handler and the message controller.
