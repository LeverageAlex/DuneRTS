Detailed Description

This class defines and specify all command line argument options for the server

Therefore this class use the CommandLine-package stores variables for every possible argument. Furthermore

  • the parameter name in the cli
  • the alias
  • whether the argument is reqiured
  • a help text is defined for every argument

The arguments, which are defined in this class, can be found in the Version 0.3 of the "Standidisierungsdokument
