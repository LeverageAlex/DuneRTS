Detailed Description

Represents a "Deserts of Dune"-Party, which is used for playing the game, so executing the game logic as well as the game start and ending.

Therefore this class stores all information about the party (identifier of this party) and the connected clients to this party. If two players are connected, the party can be prepared and started with this class. Afterwards it executes all game phases via the GameData.gameObjects.RoundHandler. Furthermore this class regularly check for the winning condition and can end the party or launch the end game phase.
