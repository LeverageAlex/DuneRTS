Detailed Description

The Logger-class offers the possibility to create and configure the global logger.

The logger is from the Serilog-Package and is used for logging informationen to the console or to files. It can be used with Log.<level>(...);
Because this class only should create and configure the (global acessable) logger, there are static methods for configuring the logger on different ways.
