Detailed Description

Base class for the connection handler, which implement the websockts and handle all the connection related tasks

Therefore this class stores all information (adress, port, protocal) needed to identify the connection endpoint to connect to or to offer and provide this url. Furthermore it force the inheriting classes to implement callbacks for the different events, which can occur when using websockts:

  • opening a connection
  • closing a connection
  • receiving a message via the network
  • notice an error
