Detailed Description

Represents one party, an ai client is playing.

Therefore it stores all informationen, necessary for preparing and playing one party und provides methods for the preparation and the execution of a game, so methods, which trigger the appropiate events. The necessary information are:

  • the party configuration entries
  • the map and characters, so the entire "world"
  • the chosen great house
